Programming & Database Work Sample

Stored Procedure – Shift Earnings Report

MySQL · SQL Logic · Aggregation & Conditional Logic

CREATE PROCEDURE sp_CalculateShiftEarnings(
    IN emp_id_param INT,
    IN shift_date DATE
)
BEGIN
    -- Calculate total sales and tips for specific shift
    SELECT
        e.employee_name,
        COUNT(t.transaction_id) AS total_appointments,
        SUM(t.tip_amount) AS total_tips,
        CASE
            WHEN SUM(t.tip_amount) > 100 THEN 'High Performance'
            ELSE 'Standard'
        END AS performance_rating
    FROM employees e
    JOIN transactions t ON e.id = t.emp_id
    WHERE e.id = emp_id_param
      AND DATE(t.created_at) = shift_date
    GROUP BY e.id;
END;

This procedure calculates an employee’s performance for a particular shift, using joins, aggregation, and conditional logic. It is part of a larger tip and sales tracking system.

Core Competencies: Database Logic, Problem Solving, SQL Proficiency
