Data Analysis Script (Python)

Python · CSV Processing · Reporting

To expand my skills beyond web development, I created a Python script that reads data from CSV files and generates simple reports. The script cleans the data, calculates summary statistics (like totals and averages), and can export the results or print them in a readable format. I also experimented with basic charting options to visualize trends. 

import csv

def load_sales(path):
    with open(path, newline="") as f:
        reader = csv.DictReader(f)
        return [row for row in reader]

rows = load_sales("sales.csv")
total = sum(float(r["amount"]) for r in rows)
average = total / len(rows)

print(f"Total sales: ${total:.2f}")
print(f"Average sale: ${average:.2f}")

This project helped me practice working with files, loops, and functions in Python, and it showed me how programming can save time by automating repetitive analysis tasks.
